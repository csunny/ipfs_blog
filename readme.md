# IPFS Blog

This is a blog which use IPFS

# Quick start

## clone代码
```
git clone git@github.com:csunny/ipfs_blog.git
```

## 添加文件

```
ipfs add -r ipfs_blog
```
## 启动服务
```
ipfs daemon
```

## 预览  
```
http://localhost:8080/ipfs/QmbstVVrYr6baRjuKQzJqx34GPjNagrSXCM9HDTRvZSS7g
```
